Overview:
- To create a machine learning algorithm that can create an image from a collection of photographs.
- This can then be uploaded to an Instagram account automatically
- The bot can read how many likes the image recieves, and if it's high enough will add the image to the photo collection.
- Images will become more and more abstract, following will decide the art's direction.
